Android下利用Charles监听Https

Android手机利用Charles监听Https请求

1,安装证书

方法1 (Android 4.0以上):

Android 4.0以上的手机支持用户安装证书。

1,下载证书到手机上

2,设置->安全->从存储设备安装 即可,安装时需要设置锁屏PIN或密码

方法2 (需要Root)

手机root之后

1,直接将证书拷贝到：etc/security/cacerts

2,设置->安全->受信任的凭据 ，在系统tab找到你安装上的证书，然后选中即可。这样是以系统证书存在的。

参考Android下安装证书的方法

2,设置Charles

打开Charles -> Proxy -> Proxy Settings -> SSL

勾选Enable SSL proxying

Locations添加想要监听的域名和端口，比如 www.ihaveu.com, 443

以上

附:如何设置手机代理

  • 在手机上WLAN中找到链接的网络
  • 设置网络
  • 高级选项
  • 填写PC的ip
  • 端口号通常默认 8888，否则需要到Charles中找到对应监听的端口号
  • 保存即可

如何重新编辑jar包

今天遇到一个问题，为了使用Monkey Talk作测试，需要把项目转换成AspectJ的项目，但是转换完成之后，编译项目的时候遇到了一个冲突,意思是说aspectj的某个jar包和项目中的一个jar包的about.html文件冲突了。因为about.html文件没什么用，就打算直接把它从jar包中删掉。

详细解决方法：

  1. 将jar包用解压工具解压，把解压后的about.html文件删掉
  2. 利用命令 jar cvf jar_name.jar ./* 重新打包

命令说明： jar cvf jar包名称 需要打包的文件

实际上就是利用jdk提供的命令重新将文件打包而已.

参考文档： jar 命令文档

Vim快捷键

记得是在刚上班的时候就开始用Vim，不过到现在为止感觉还是没有完全将Vim的潜力完全挖掘出来。
这里主要记录一下常用的快捷键，方便查找。

编辑

  • i: 在当前字符的左边插入
  • I: 在当前行首插入
  • a: 在当前字符的右边插入
  • A: 在当前行尾插入
  • o: 在当前行下面插入一个新行
  • O: 在当前行上面插入一个新行
  • d: 删除.删除和查找,方向，重复次数方向都可以组合操作
  • dd: 删除当前
  • D: 删除当前为止到行尾

查找替换

  • [范围]s/[源字符]/[目标字符]/[g]: 替换字符
  • s/[源字符]/[目标字符]: 替换字符,当前行
  • %s/[源字符]/[目标字符]: 替换字符,整个文件范围
  • 1,20 s/[源字符]/[目标字符]: 替换字符,1到20行
  • /[字符表达式]: 查找字符
  • /admin: 查找admin
  • /a.*in: 查找类似admin,aaxxxin,axxxxin,. 匹配单个字符,*匹配任意字符 类似正则表达式，

光标定位

  • k: 上
  • j: 下
  • h: 左
  • l: 右
  • gg: 到文件首
  • G: 到文件尾
  • ^: 到行首
  • $: 到行尾
  • w: 移动到下一个单词
  • b: 移动到上一个单词
  • %: 跳到匹配的括号处（”{ }””[]””()”）
  • [num]G: 移动光标到指定的行(num),比如: 10G就是到第10行
  • [num]gg: 移动光标到指定的行(num),比如: 10G就是到第10行
  • [num]:: 移动光标到指定的行(num),比如: 10G就是到第10行
  • &lt;control-i, tab&gt;:跳到下一个跳点(Jump Point)
  • &lt;control-o&gt;:跳到上一个跳点
  • H: 移动光标到屏幕上面
  • M: 移动光标到屏幕中间
  • L: 移动光标到屏幕下面
  • &lt;control-F&gt;: 向下移动一屏。
  • &lt;control-B&gt;: 向上移动一屏。
  • *: 读取光标处的字符串，并且移动光标到它再次出现的地方。
  • \#: 和上面的类似，但是是往反方向寻找。
  • ma: 在当前光标的位置标记一个书签，名字为 a。书签名只能是小写字母。你看不见书签的存在，但它确实已经在那里了。
  • `a: 到书签 a 处。注意这个不是单引号，它一般位于大部分键盘的 1 的左边。
  • `.: 到你上次编辑文件的地方。这个命令很有用，而且你不用自己去标记它
  • /text: 从当前光标处开始搜索字符串 text，并且到达 text 出现的地方。必须使用回车来开始这个搜索命令。如果想重复上次的搜索的话，按n。
  • ?text: 和上面类似，但是是反方向。
