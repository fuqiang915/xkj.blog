Adb Shell 命令总结

目录

Shell

  • 登入shelladb shell //运行远端shell交互
  • 运行远端shell 命令: adb shell <command>
  • 运行仿真控制台命令: adb emu <command>

进程信息相关

  • 获取手机所有进程: adb shell ps -x
    • 利用grep筛选关键字：adb shell ps -x | grep qiyi
    • VSS - Virtual Set Size 虚拟耗用内存（包含共享库占用的内存）
    • RSS - Resident Set Size 实际使用物理内存（包含共享库占用的内存）
    • PSS - Proportional Set Size 实际使用的物理内存（比例分配共享库占用的内存）
    • USS - Unique Set Size 进程独自占用的物理内存（不包含共享库占用的内存）
  • 设备信息：cat /system/build.prop | grep "product"
  • CPU信息：/proc/cpuinfo和/proc/stat
  • 内存：/proc/meminfo adb shell dumpsys meminfo

文件导入导出

  • 导入: adb push <local> <remote>
    • adb push <local_file_path> /sdcard/

  • 导出: adb pull <remote> [<local>]

    • adb pull /sdcard/ <local_file_path>

  • 文件同步: adb sync [<directory>] //只有发生改变时从主机拷贝到设备

日志

  • 浏览日志: adb logcat [<filter-spec>]

应用卸载安装

  • 拷贝文件包到设备并安装: adb install [-l] [-r] [-s] <file>
  • 卸载程序包，-k意味着保留数据和缓存: adb uninstall [-k] <package>
  • 返回所有的bugreport信息: adb bugreport
  • 转发套接字连接: adb forward <local> <remote>

连接设备

  • 阻塞直到设备上线: adb wait-for-device
  • 启动服务： adb start-server
  • 关闭服务：adb kill-server
  • 列印offline,bootloader,device信息: adb get-state
  • 获取序列号：adb get-serialno
  • 连续列印设备状态:adb status-window
  • 重装载/system分区: adb remount
  • 重启: adb reboot [bootloader|recomry]
    • adb reboot-bootloader
  • 获取权限：adb root
  • 重启adb重新监听usb：adb usb
